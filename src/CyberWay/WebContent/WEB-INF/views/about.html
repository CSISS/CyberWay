Introduction of CyberWay

Project Description

  • This project will build a production system of systems integrating EarthCube building blocks, particularly BCube, CyberConnector, CHORDS, and GeoWS, to advance geosciences research through improved interoperability and integration across diverse polar science and atmospheric science research, methods, and resources.
  • Named CyberWay, the system is built through connecting Earth observations, polar science outcomes, and climate model outcomes via the EarthCube architecture. The system itself will also become an element of the EarthCube infrastructure.

Science Use Case

The scientific driver is to advance multidisciplinary science research, specifically climate science and polar science, by integrating capabilities of EarthCube building blocks. The following are the five active NSF GEO science projects to be supported by CyberWay.

four cases:

  • Cross-discipline collaboration: Associate model outputs with Earth observations and reanalyses
  • Compare model outputs with Earth observations (EO)
  • Streamlining Earth observations for ASR
  • Enabling discovery of tropical cyclones (TC) by cataloging with CyberWay

System of Systems

CyberWay will integrate the capabilities and systems of primarily four existing EarthCube building blocks, i.e. BCube (a broker framework for geosciences)[23], [24], GeoWS (Geoscience Web Services)[25], CHORDS (EarthCube's Cloud-Hosted Real-time Data Services for the Geosciences)[26], and CyberConnector[27].
